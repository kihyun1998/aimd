# Changelog

## [v1.3.0] - 2025-02-14

### 추가됨
- 파일 분할 기능
  - 대용량 출력 파일 자동 분할 기능 추가
  - maxsize 옵션을 통한 파일 크기 제한 설정
  - 기본 분할 크기 10MB 설정
  - 분할 파일 자동 넘버링 (예: CODE1.md, CODE2.md)
- FileSplitter 인터페이스 도입
  - 파일 분할 로직 모듈화
  - SOLID 원칙 준수 설계

### 기술적 특징
- 파일 시스템 처리 개선
- 메모리 사용 최적화
- 클린 아키텍처 적용
- SOLID 원칙 기반 리팩토링
